simpLoad

A simple way to indicate AJAX-loading

Apply simpLoad to the whole page

View Code 

$('#button').click(function(){

    //Start Simpload
    $('body').simpLoad('start');

    //Stop Simpload
    setTimeout(function(){
        $('body').simpLoad('stop');
    }, 2000);

});
